# resume-refiner

> 不重写你的人生。  
> 只把已有简历改得更贴近目标岗位。

`resume-refiner` polishes an existing resume for a target JD without inventing experience.

## Use when

- You already have a resume
- You have a target JD or role direction
- You want better wording, priority, and evidence strength

## Output

- Resume diagnosis
- JD relevance priorities
- Suggested edits
- Sample bullet rewrites
- Missing facts to confirm

## Next step

Use `$experience-translator` to turn the strongest revised experience into interview language.
