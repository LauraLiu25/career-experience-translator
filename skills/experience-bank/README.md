# experience-bank

> 每次准备过的经历，都应该变成资产。

`experience-bank` stores and reuses compact experience notes across roles and JDs.

## Use when

- You want to save an experience note
- You want to list previous experiences
- You want to find which saved experience fits a new JD
- You want to adapt an old experience to a new role

## Output

- Experience note table
- Tags
- Best matching saved experiences
- Handoff to `$experience-translator` or `$resume-refiner`
