# role-fit-check

> 不是判断你能不能投。  
> 是帮你找到哪些经历可以支撑这个岗位。

`role-fit-check` compares a target JD with an existing resume/profile and identifies direct evidence, transferable evidence, and honest gaps.

## Use when

- You have a JD and a resume
- You want to know which experiences are worth using
- You are worried your background is not "vertical" enough

## Output

- JD signal table
- Resume evidence mapping
- Transferable fit points
- 1-2 experiences worth refining next
