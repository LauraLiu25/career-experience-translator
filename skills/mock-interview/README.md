# mock-interview

> 写得出来，不等于说得出来。

`mock-interview` runs a focused interview drill around one real prepared experience.

## Use when

- The user has a prepared experience
- The user wants realistic follow-up questions
- The user needs to practice speaking naturally

## Output

- One question at a time
- Focused feedback
- Follow-up questions about decisions, difficulty, feedback, and improvement

## Best used after

`$experience-translator`
