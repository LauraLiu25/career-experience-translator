# story-doctor

> 你以为经历很普通。  
> 可能只是还没把故事问出来。

`story-doctor` helps mine useful story material from ordinary experiences by asking about conflict, choices, change, and impact.

## Use when

- The user says "I have nothing to say"
- An experience feels too plain
- A resume bullet lacks detail
- Interview answers sound empty

## Output

- Targeted questions
- Story element list
- Hidden difficulty or decision points
- Handoff to `$experience-translator` or `$resume-refiner`
